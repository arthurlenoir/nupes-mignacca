# Je ne pourrai pas voter les 12 ou 19 juin

#### Comment faire une procuration

Vous pouvez établir à l’avance une procuration pour qu’une personne de confiance vote à votre place. Celle-ci peut habiter n’importe où (elle n’a pas besoin d’être dans la même commune), mais **elle devra se rendre dans votre bureau de vote pour voter.**

Si vous n’avez personne à qui confier votre procuration, vous pouvez utiliser notre outil en ligne pour vous mettre en relation avec une personne de confiance qui acceptera de voter pour vous :

<Button
  as="a"
  href="https://actionpopulaire.fr/procuration/donner-ma-procuration/"
  target="_blank"
  variant="primary"
>
  Remplir la demande
</Button>

<br />

Une fois la personne trouvée et d’accord pour prendre votre procuration, c’est à vous de faire la procédure pour la désigner. Il existe une procédure en ligne pour établir votre procuration sur le site maprocuration.gouv.fr.
